Development

Requirement

  • npm

Setup

npm install

Create a new tailwind.css

To compile a new tailwind css file run the following command:

npm run prod

Release management

Prerequisites

  • Write access to the repositoty
  • Github CLI
  • npm
  • make

New release

To create a new release set a new version in ./VERSION and run make release.
